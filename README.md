# Mobile App Wireframing

## Project Overview

This project demonstrates the complete UI/UX planning process for a Food Delivery Mobile Application.

The project includes:

- User Research
- User Persona
- User Flow
- Low Fidelity Wireframes
- Design Thinking Process

---

## Tools Used

- Figma
- GitHub
- Markdown

---

## Features

- Login
- Sign Up
- Home Screen
- Search Food
- Food Details
- Cart
- Checkout
- User Profile
